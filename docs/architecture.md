## Data View

In this section the data saved on database is described.

- Model District

| Data     | Type          | Required |
| -------- | ------------- | -------- |
| name     | string        | Yes      |
| type     | string        | Yes      |
| geometry | polygon field | No       |
| CD_MUN   | string        | No       |
| NM_MUN   | string        | No       |
| SIGLA_UF | string        | No       |
| AREA_KM2 | integer       | No       |

Example:

```
    "name": "Brasília"
    "type": "Feature",
    "geometry": "Polygon(((0.0, 0.0), (0.0, 50.0), (50.0, 50.0), (50.0, 0.0), (0.0, 0.0)))",
    "CD_MUN":"5300108",
    "NM_MUN":"Brasília",
    "SIGLA_UF":"DF",
    "AREA_KM2": 5760.784
```

- Model State

| Data       | Type          | Required |
| ---------- | ------------- | -------- |

| type       | string        | Yes      |
| geometry   | polygon field | No       |
| CD_UF      | string        | No       |
| POPULATION | integer       | No       |

